# promise-portal

let you use react portal in vue, and with promise

## packages

- promise-portal

  > source code

- playground

  > develop demo

## Install

```bash
// pnpm
pnpm add promise-portal -D

// npm
npm install promise-portal -D

// yarn
yarn add promise-portal --D
```

## Develop

```bash
pnpm run dev
```

then open `http://localhost:9001/`

## Build

```bash
pnpm run build
```
